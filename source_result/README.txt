A: How to compile:
Open terminal, go into java_code file, then:
>> make
>> java dm_project

Note: Input train_all_txt.txt file has been put in java_code file. 
      Make sure it can run correctly, please do not remove the input train_all_txt.txt file.
      Then the output.txt will be created in java_code file.
